🪨✂️📜 ROCK - PAPER - SCISSORS GAME

This is a Rock-Paper-Scissors Game built with HTML, CSS, and JavaScript.
The player competes against the computer (robot). Each round, the robot makes a random choice among rock, paper, or scissors. The player selects their own sign from the dropdown menu and clicks “Start” to play.

The program automatically determines the winner according to the rules:

Rock beats Scissors

Scissors beats Paper

Paper beats Rock

A live scoreboard keeps track of both the player’s and the robot’s scores.
The game dynamically updates visuals to show both sides’ chosen signs and displays a result message such as “You won! 🎉”, “You lost! 😢”, or “Draw!”.

🧩 Features

🎯 Simple and interactive UI

🤖 Random robot selection logic

🔄 Dynamic score update system

🧠 Result calculation with clean JavaScript conditions

🎨 Modern design using CSS (custom font + soft color palette)

🧠 Technologies Used

HTML5 – basic structure

CSS3 – layout, colors, and animations

Vanilla JavaScript (ES6) – game logic and DOM manipulation

🚀 How to Play

Choose your sign (Rock, Paper, or Scissors) from the dropdown menu.

Click the Start button to play a round.

The robot will randomly select its move.

See who wins and track the scores at the top.

🪨✂️📜 TAŞ - KAĞIT - MAKAS OYUNU

Bu proje, HTML, CSS ve JavaScript kullanılarak oluşturulmuş bir TAŞ – KAĞIT – MAKAS oyunudur.
Kullanıcı, bilgisayara (robota) karşı yarışır. Oyuncu menüden bir sembol seçip “Start” butonuna bastığında robot rastgele bir seçim yapar.

Sonuç kurallara göre belirlenir:

Taş makası yener

Makas kağıdı yener

Kağıt taşı yener

Her tur sonunda skorlar otomatik güncellenir ve sonuç metni ekranda görünür (“Kazandın 🎉”, “Kaybettin 😢” veya “Berabere”).

🎯 Özellikler

Kullanıcı dostu, basit arayüz

Robotun rastgele seçim algoritması

Canlı skor takibi

Dinamik görsel değişim (aktif img’lerin değişmesi)

Renkli, modern CSS tasarımı

🧠 Kullanılan Teknolojiler

HTML5 → sayfa yapısı

CSS3 → tasarım, font ve düzen

JavaScript (Vanilla JS) → oyun mantığı ve DOM kontrolü

🕹️ Nasıl Oynanır

Menüden “Stone”, “Paper” veya “Scissors” seç.

“Start” butonuna bas.

Robot rastgele bir seçim yapacak.

Kazanan ekranda gösterilecek, skor güncellenecek.
